### Features
- **Lead Dashboard**: Full project control, task assignment, code review approval
- **Member Dashboard**: View assigned tasks, submit code for review
- **Start Screen**: Create new projects or join existing sessions
- **User Profiles**: Display GitHub avatar and username
- **Session Tokens**: Shareable invite tokens for team collaboration
- **Real-time Sync**: All changes broadcast to team members instantly

### Technical
- Built with TypeScript and VS Code Extension API
- WebSocket communication for real-time features
- PostgreSQL for data persistence
- Google Gemini 1.5 Flash for AI capabilities
- Axios for HTTP requests
- React components for UI (future enhancement)
